## 데이터의 이해

* 암묵지와 형식지의 상호작용 관계 : '공통화 -> 표출화 -> 연결화 -> 내면화'
* 반정형 데이터 : 데이터 내부에 메타 데이터를 갖고 있으며 일반적으로 파일 형태로 저장됨
* 지식 : 상호 연결된 정보 패턴을 이해하여 이를 토대로 예측한 결과물
* Cinematch 시스템 : 넷플릭스에서 개발한 영화 추천 알고리즘
